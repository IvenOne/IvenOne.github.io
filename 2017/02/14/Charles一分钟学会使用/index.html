Charles一分钟学会使用

  • charles 使用 抓包

##简介

Charles 网络数据包截取工具。在移动开发时的利器，在mac OS 中推荐使用，windows 可以考虑fiddle. Charles最新破解版下载

##功能

  1. 截取数据包 http and https
  2. 重发请求
  3. 修改请求参数
  4. 修改返回数据
  5. 模拟网速

使用

  1. 设置代理，如图

  2. 使用

    可以在Structure和Sequence视图之间进行切换。

    使用时一定要注意必须在record的状态下,否则不会有任何抓取功能。

    • 使用repeat功能 （repeat advance）

      此功能可以用来重复发送请求

    • 修改网络请求

      选中抓取的连接右击，点击Edit,即可

      当然可以选择cancel进行撤销，以及revert进行还原

    • 使用throttling进行网速测试

      方法：proxy—>throttle setting ——> 设置网速进行添加。

    • 修改返回数据

      修改返回数据有几种方式：

      1. Map 功能适合长期地将某一些请求重定向到另一个网络地址或本地文件。
      2. Rewrite 功能适合对网络请求进行一些正则替换。
      3. Breakpoints 功能适合做一些临时性的修改。

      在这里我们说下经常使用的breakpoint功能。

      方法是双击要break的连接，然后单击breakpoints，在运行一次此连接就可以调试了，时间不能太久否则会出现timeout提示。另外一点可以在setting中设置只在request 中或者response中or Both of them使用 调试。最后点击execute就OK了。
